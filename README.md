# RAG Workshop

Learn Retrieval-Augmented Generation from prototype to implementation.

## Getting started

```bash
git clone https://github.com/izaakrogan/rag-workshop.git
cd rag-workshop
```

Then open Exercise 1 and follow the README.

## Exercises

| Exercise | Description |
|----------|-------------|
| [Exercise 1](./exercises/01-simple-rag/) | Build a simple RAG prototype |
| [Exercise 2](./exercises/02-exploring-embeddings/) | Explore embeddings and cosine similarity |
| [Exercise 3](./exercises/03-rag-from-scratch/) | Rebuild the RAG pipeline from scratch with no vector db |
